* sysctl.conf
- elastic-searchコンテナ起動時に発生するエラー対策
- ip_forwardingが無効になる問題

- docker01, 02, 03
#+begin_src sh
sudo echo "vm.max_map_count = 262144" >> /etc/sysctl.conf
sudo echo "net.ipv4.ip_forward = 1" >> /etc/sysctl.conf
#+end_src
